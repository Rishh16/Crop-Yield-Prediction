What It Does

This dashboard supports crop planning by combining season and land area inputs with machine learning predictions derived from historical agricultural data.

It also integrates real-time field telemetry from ESP32/Arduino devices (soil moisture, temperature, humidity) for monitoring and decision support.

Outputs are presented as crop suitability percentages to help farmers quickly identify promising crop options for the selected season.

How It Works

1

Enter Inputs

Select season and enter cultivated area in acres. Sensor values can stream automatically from IoT hardware.

2

Model Processing

The backend evaluates crop options using trained ML models and computes normalized suitability percentages.

3

Recommendation Output

The dashboard returns the best crop for the season and additional crop percentages for comparison.
